# 🎓 Front-End Google Clone

---

A simple front-end recreation of the Google homepage layout.
This project was built using Visual Studio Code as a beginner-friendly exercise to improve my HTML and CSS skills.

The design focuses on replicating:

🔍 The central search bar

🎯 Navigation elements

🧭 Clean and minimal layout similar to Google

Navigation links have been customised to redirect to my personal platforms, such as LinkedIn and GitHub.

## 📌 The Process

In preparation for my future career, I have started building websites on a monthly basis to showcase my experience and improve my understanding of new technologies.

As a beginner, I required guidance and a basic framework to help me understand how to structure and style a website. This included:

YouTube tutorials
AI integration for efficiency, functionality, and quality

This project focused specifically on improving my understanding of layout, spacing, and visual design using HTML and CSS.

Like any learning process, this project involved trial and error, helping me become more comfortable with structuring web pages and styling elements.

There are many opportunities to expand this project, such as adding search functionality or improving responsiveness across devices. I plan to continue improving and refining this project as I gain more experience.

Overall, I am very satisfied with the result as it strengthens my foundation in front-end development and represents another step forward in my coding journey.
